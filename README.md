# uzleti-intelligencia
Üzleti Intelligencia megajánlott jegyes feladat - Kriptovaluta és részvény adatok elemzése

A pénzügyi piacok napjainkban jelentős szerepet töltenek be a gazdaságban, legyen szó digitális
eszközökről vagy hagyományos részvényekről. A házi feladat kriptovaluta és részvény adatokat dolgoz
fel annak érdekében, hogy hasznos és könnyen értelmezhető információkat nyerjünk, például az
árfolyamok alakulásáról, a legnagyobb napi változásokról, a forgalomról és a volatilitásról. Az
elemzések célja, hogy áttekintést adjanak a piacok mozgásairól és a különböző eszközök
teljesítményéről.
